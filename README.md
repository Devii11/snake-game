# Snake Game

A fun snake game built with HTML, CSS, and JavaScript featuring multiple food types and special abilities.

## Features

- **Red Food**: +10 points and makes the snake grow
- **Blue Food**: Cuts the snake size in half without losing points
- **Yellow Food**: 50% speed boost for 1 minute
- **Wraparound**: Snake wraps around edges - no walls!
- **Responsive Controls**: Use arrow keys to control the snake

## How to Play

1. Open `snake_game.html` in your web browser
2. Use arrow keys to control the snake
3. Eat food to gain points and special abilities
4. Avoid hitting yourself
5. Press SPACE to restart when game over

## Game Mechanics

- Snake moves continuously in the current direction
- Speed boosts from yellow food last 1 minute
- Blue food appears randomly and cuts snake length in half
- Yellow food appears frequently for speed advantages
- Snake wraps around screen edges for continuous play

## Technologies Used

- HTML5 Canvas for game rendering
- Vanilla JavaScript for game logic
- CSS for styling and layout
